# Calculadora de distancia

[Ver proyecto completo en microbit\.org](https://microbit.org/es-es/projects/make-it-code-it/distance-calculator)

Para codificar este proyecto tú mismo, haz clic derecho en el fondo del área de trabajo y elige ‘delete all blocks’, luego sigue el video de codificación a continuación. Puedes encontrar los bloques forever y on start en la sección Basic.  
  


### ¿Qué es?

Utiliza tu BBC micro:bit para medir distancias mientras caminas.

https://www.youtube.com/watch?v=Uzm4Kss4AOA

### Qué aprenderás

Este proyecto práctico muestra cómo utilizar la multiplicación y las variables para medir distancias.

### Cómo funciona

* Este programa calcula las distancias multiplicando la longitud de los pasos por el número de pasos que das. Utiliza una longitud media de paso infantil de 0,6 metros.
* Recorre cuidadosamente la distancia que deseas medir, presionando el botón A cada vez que tomes una medida.
* El programa funciona incrementando la variable "cuenta pasos" en uno cada vez que se pulsa el botón A.
* Pulsa el botón B para ver la distancia que has recorrido en metros. El micro:bit lo calcula multiplicando la variable ''cuenta pasos'' por la longitud del paso.
* Si quieres saber cuántos pasos has dado, pulsa el logotipo táctil de la parte frontal de tu micro:bit.
* Reinicia el micro:bit pulsando el botón de reinicio de la parte posterior.
